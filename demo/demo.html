Vue Feature Flipping

Service isEnabled()

<div v-if="isEnabled('FEAT')">
<div v-if="isEnabled('FEAT', false)">
<div v-if="isEnabled('FEAT', true)">

Render DOM (like v-if)

<div v-feature-flipping="'FEAT'">
<div v-feature-flipping.not="'FEAT'">
<div v-feature-flipping.default="'FEAT'">

Apply class (like v-bind:class)

<div v-feature-flipping:class="{ key: 'FEAT', value: 'enabled' }">
<div v-feature-flipping:class.not="{ key: 'FEAT', value: 'enabled' }">
<div v-feature-flipping:class.default="{ key: 'FEAT', value: 'enabled' }">

Apply style (like v-bind:style)

<div v-feature-flipping:style="{ key: 'FEAT', value: { color: 'green' } }">
<div v-feature-flipping:style.not="{ key: 'FEAT', value: { color: 'green' } }">
<div v-feature-flipping:style.default="{ key: 'FEAT', value: { color: 'green' } }">
